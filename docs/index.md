# test-poetry-app

A short description of the project

## Overview

This documentation covers the test-poetry-app application.

## Installation

```bash
pip install test_poetry_app
```

Or, if you use Poetry:

```bash
poetry add test_poetry_app
```


## Usage

Run the application:

```bash
test_poetry_app
```

Or run as a module:

```bash
python -m test_poetry_app
```


## Quick Start

```bash
pip install test-poetry-app

test_poetry_app
```
