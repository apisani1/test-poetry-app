## Overview

This documentation covers the test-poetry-app application.

## Installation

```bash
pip install test-poetry-app
```

Or, if you use Poetry:

```bash
poetry add test-poetry-app
```

## Usage

Run the application:

```bash
test_poetry_app
```

Or run as a module:

```bash
python -m test_poetry_app
```

## Quick Start

```bash
pip install test-poetry-app
```
